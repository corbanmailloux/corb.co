Because I didn't feel like paying to renew it, the domain alittleteapot.im has now expired. I've hosted the code from the site at tea.corb.co.

alittleteapot.im is a beautifully animated teapot, made with CSS animations. I initially bought the domain because I was thinking of fun Java package names. If I were to make a package called ShortAndStout, the package name could be Im.ALittleTeapot.ShortAndStout.

To get the lettering just right, it uses animate.css, lettering.js, and textillate.js.

The beautiful teapot animation, on the other hand, was made by me.

The result is available at tea.corb.co, and the code is available on GitHub.
